+----------------------------------+
 VeryIDE Bee 应用平台简介
+----------------------------------+

	VeryIDE Bee 是 VeryIDE 推出的一套通用的 Web 应用平台系统。 
	
	用它来配合论坛 定制表单 或是 管理多个网站的广告 再或是 制作专题页面 都将是易如反掌。更多功能请安装后试用。

+----------------------------------+
 VeryIDE Bee 商业用途说明
+----------------------------------+

	VeryIDE Bee 虽然可以 (非商业用途) 免费使用，但是我们为商业 (授权) 用户提供了更完善的功能和技术支持服务。
	
	商业用户 功能/服务 目录请见 http://www.veryide.com/market.php

+----------------------------------+
 VeryIDE Bee 应用平台环境需求
+----------------------------------+

	1. 可用的 www 服务器，如 Apache、Zeus、IIS 等  
	2. php 5.0 及以上 
	3. MySQL 3.23 及以上，仅针对 MySQL 版

+----------------------------------+
 VeryIDE Bee 平台安装说明
+----------------------------------+
	
	+----------------------+
		VeryIDE 安装步骤：
	+----------------------+

		1.请严格按照本说明进行一步步安装，否则可能安装不成功
		
		2.推荐使用二级域名来安装本程序，例如：app.veryide.com
		
		3.请将目录下的所有文件上传至服务器
		
		4.设置目录属性（windows 服务器可忽略这一步）
			以下这些目录需要可读写权限 
			./attach/ 
			./cache/ 
			./config/ 
			./static/ 
			./data/backup 
			./data/fixbug 
			./data/update 
			./data/install 
		
		4.创建与当前版本编码一致的数据库
		
		5.访问你上传的地址，开始安装
		
		5.安装完成后，请记得设置 “超级管理员” 和普通用户组权限【重要】

	
	+----------------------+
		VeryIDE 系统配置：
	+----------------------+

		1.数据库及目录配置：config/config.php
		
		2.远程附件配置：config/attach.php
		
		3.全局及插件配置：config/setting.php

	+----------------------+
		VeryIDE 模块的配置
	+----------------------+

		1. 模块存放地址 ./module
		2. 参照模块配置文件 /module/*/config.php 中的注释进行配置
		2. 部分模块可能需要以下PHP函数库支持：
			GD、XML、iconv、mcrypt
	
	+----------------------+
		VeryIDE 通行证整合（商业版功能）：
	+----------------------+

		1.Discuz：module/discuz/setting.php

		1.DiscuzX：module/discuzx/setting.php
		
		3.UCenter：module/ucenter/setting.php
		
			API地址:
			http://[localhost]/module/ucenter
		
		4.PHPWind：module/phpwind/setting.php
		
		对于 Discuz 论坛可以使用 Discuz 方式或 UCenter 方式，但还是推荐使用 Discuz
